# 車庫証明かんたん作成（静岡県東部 ・ 沼津・三島・富士・伊豆ほか 対応）

行政書士の方が、**スマホ1台で車庫証明（自動車保管場所証明）の書類一式**を作成できるツールの **静岡県東部版** です。
入力は1回だけ。申請書・自認書／使用承諾証明書・所在図／配置図・提出チェックリストへ自動転記します。

`index.html` の1ファイルだけで動きます。インストール不要・オフライン動作・**個人情報は端末内のみで処理**（外部送信なし）。

> 🌐 **公開URL（スマホで開く）: https://atsunaricoda-maker.github.io/shako-shomei-tobu/**
> このURLをスマホのブラウザで開き、「ホーム画面に追加」でアプリになります（PWA）。
>
> 中部版（静岡市・志太榛原）: https://atsunaricoda-maker.github.io/shako-shomei-chubu/
> 西部版（磐田・浜松ほか）: https://atsunaricoda-maker.github.io/shako-shomei-iwata/

## 特長

- **1回入力 → 全書類に自動転記**（申請書 or 届出書／自認書 or 承諾書／所在図・配置図／チェックリスト）
- **普通車／軽自動車に対応**：種別を選ぶと様式・手数料（2,400円↔無料）・文面が自動で切替
- **軽の届出義務を地域で自動判定**：東部で軽の車庫届出が必要なのは実質**三島・沼津・富士・富士宮**の4市のみ。沼津（旧戸田村・清水町）・富士（旧富士川町）・富士宮（旧芝川町）は管轄内に**届出不要の地域**があり、その注記も自動表示。御殿場・裾野・伊豆方面・賀茂方面は届出不要
- **静岡県東部10警察署をプリセット**：提出先を選ぶだけで宛先・所在地・電話が全書類に反映
- **航空写真でなぞって配置図**：Googleマップの航空写真スクショを取り込み、駐車枠を実際の位置にドラッグ＋サイズ・回転で合わせて作図（透かし表示）
- **スタンプで配置図**：「保管場所」「自宅」「道路」「北」「出入口」などの部品を置いてドラッグで移動・サイズ変更
- **地図をそのまま所在図に**：取り込んだ地図画像に「本拠」「車庫」の印・直線・距離を記入して所在図にする
- **配置図をSVGで自動作図**（航空写真を使わない場合） / **本拠⇔保管場所の2km判定** / 車両がスペースに収まるか判定 / 和暦変換
- **案件の保存・読み込み**（端末内）/ **印刷・PDF保存**（A4・コンビニ印刷可）
- **作った案件を事務所へ“送信”**（印刷の代わりに使える）：⚙に「お店名・事務所メール」を設定 → ステップ⑨の **📤 事務所へ送信** で案件データを**リンク化してメール/LINE送信**。受け取った事務所は**リンクを開くだけで入力済み**で開き、PCで印刷・転記。**サーバー不要・あなたの維持ゼロ**
- **自動更新**：新しい版を公開すると画面下に「🔄 新しい版があります → 更新」が出て、**ワンタップで最新化**（入力中に勝手にリロードしない／古い版が残り続ける問題を解消）

## 軽自動車の保管場所届出が必要な地域（東部）

静岡県警の公式一覧より、東部で**届出が必要なのは次の4市のみ**（カッコ内は管轄内でも**届出不要**の地域）。

- **三島市**（全域義務）
- **沼津市**（ただし旧戸田村＝戸田地区は不要。駿東郡清水町も不要）
- **富士市**（ただし旧富士川町＝富士川地区は不要）
- **富士宮市**（ただし旧芝川町＝芝川地区は不要）

御殿場市・裾野市・伊豆の国市・伊豆市・熱海市・伊東市・下田市、および小山町・長泉町・函南町・賀茂郡5町は**いずれも届出不要**（車庫の確保自体は必要）。

## 静岡県共通の仕様（2026年6月時点・公式確認済み）

- 手数料 **2,400円**（保管場所証明手数料・静岡県**収入証紙**で納付）。**県内全署共通**（市が変わっても提出先の警察署が変わるだけ）
  - ※令和8年（2026年）3月31日までは2,200円でしたが、**4月1日改定で2,400円**（本ツールは2,400円を表示）
- **保管場所標章は2025年4月1日に廃止**（車庫シール・標章交付手数料なし）。軽自動車も届出のみ・手数料無料
- 本拠と保管場所は**2km以内**／押印は2021年〜不要

## 案件データの送信（現場入力 → 事務所）— サーバー不要

「車屋さんなどがタブレットで入力 → 行政書士事務所へ送る」運用ができます。バックエンド（サーバー）を一切持たない**リンク受け渡し方式**です。

1. 送る側：右上 **⚙** で「**お店名（送信元）**」と「**事務所メール（送信先）**」を1回設定
2. 入力後、ステップ⑨の **📤 メールで送信**（または **🔗 リンクをコピー** でLINE等）
3. 受け取る側（事務所）：届いた**リンクを開くだけ**で、このアプリが**入力済み**で開く → 確認 → **PCの「印刷 → PDFに保存」**で書類化／公式様式へ転記

- **サーバー・アカウント不要／作者の維持はゼロ**（多数へ配っても増えない）。各利用者の設定は「メールを1回入れるだけ」なので**売り切り配布に向く**
- 仕組み：入力内容をURLの `#` 以降に符号化して埋め込むだけ（毎回ちがうリンク＝中身が違うから）。保存先を持たないので**期限切れ・管理不要**
- リンクの中に案件データが入っています（**符号化であって暗号化ではない**）。**身内の連絡手段で**送ってください
- **スタンプで作図した配置図はリンクで送られます**（部品の配置データを送り、受信側で同じ図を再現）。**航空写真トレースの“写真”はリンクに含めません**（容量のため。必要なら事務所側でスタンプに描き直し）

## 事務所モード / 車屋配布モード（1つのアプリで切替）

同じアプリを、立場で使い分けられます。

- **事務所モード（既定）**：通常URL。入力・送信・受信・**印刷**すべて使える
- **車屋配布モード**：URL末尾に `?role=dealer` を付けたものを車屋さんに配る（`.../shako-shomei-tobu/?role=dealer`）。**送信のみ・印刷ボタンなし**。開くと自動で車屋モードになり端末に記憶されます
- **車屋が送ったリンクを事務所が開くと、自動で事務所モード（印刷可）**で開きます（リンクは通常URLを指すため）
- 右上 **⚙ → 利用モード** でいつでも切替・**配布用URLのコピー**ができます

> 使い方：事務所＝受け取って印刷／車屋＝入力して送るだけ。配るのは「`?role=dealer` 付きURL」だけです。

## Google Maps連携（任意・アプリ内で完結）

ヘッダーの **⚙** からGoogle Maps APIキーを入れると、スクショ不要で次がアプリ内で完結します：
- **住所から直線距離を自動計算**（2km判定も自動）
- **航空写真をアプリ内に取り込んで配置図をトレース**（ズーム調整可）
- **所在図を自動作成**（本拠・車庫のマーカー＋直線入りの地図を生成）

**キーは端末内（localStorage）にのみ保存**され、リポジトリや外部には出ません。キー未設定・オフライン時は従来の**スクショ取り込み方式に自動フォールバック**します。

**キーの作り方（Google Cloud）**
1. プロジェクト作成・**請求先（カード）登録**
2. 有効化するAPI：**Maps JavaScript API / Geocoding API / Maps Static API**
3. APIキーを作成
4. **キーを制限**：アプリの制限＝HTTPリファラー `atsunaricoda-maker.github.io/*`、API制限＝上記3つ
5. **予算アラート・割当上限**を設定（想定外課金の防止）

> ⚠️ 住所はジオコーディングでGoogleに送信されます（端末内完結ではなくなります）。地図機能はオンライン時のみ。料金は最新の無料枠・単価をGoogleでご確認ください。

## 他の署を追加・修正するには

`index.html` 冒頭の `STATIONS` 配列を編集（`kei` は軽の届出義務地域なら `"req"`、不要なら `"none"`。義務地域で管轄内に除外がある場合は `keiArea`＝対象市、`keiEx`＝除外地域の注記を付ける）。
